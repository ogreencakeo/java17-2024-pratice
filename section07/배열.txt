1. 배열이란 ... ?
- 배열의 개념은 많은 프로그래밍 언어가 다 비슷 ~> 사용법의 차이는 O ~> 한마디로 정의 ~> 큰바구니
- "변수"가 작은 과일 한 개를 담는 바구니라면, "배열"은 과일 여러 개를 담을 수 있는 큰 바구니
- 즉, 변수들의 묶음 집합체
- 특정 타입(동일한)의 데이터를 편리하게 담고 꺼내서 사용할 수 있도록 만든 큰 바구니
- 따라서, 배열은 다양한 종류의 데이터를 어떻게 담고 꺼내서 사용할 것인지에 대한 자료구조 중 하나
- 자바에서는 배열을 보다 효율적이고 편리하게 사용할 수 있는 자료구조 프레임워크를 제공 ~> 컬렉션 프레임워크(오토메틱)

2. 일상에서의 배열
- 엄마가 아이들 양말함을 만들려고 빈 우유곽 여러개를 붙여서 양말함을 만들었다면 ~> 이것이 곧 배열
- 호텔에 나혼자 투숙하러 온 손님은 "변수" 손님, 단체로 투숙하러 온 손님은 "배열" 손님

ar[0] ar[1] ar[2] ar[3] ar[4] ar[5] ar[6]  

3. 배열 모습
nums ----> nums[0] nums[1] nums[2] nums[3]

- 배열은 연속적으로 순서있게 데이터를 담고 꺼내 쓸 수 있음 ~> 인덱스는 0부터 시작
- 데이터를 남는 각각의 방에 이름을 붙이지 않아도 됨 ~> 배열 이름은 필요하지만 개별 방 이름은 필요 X
- 배열의 크기는 내가 정해서 데이터를 담을 수 있음 (장점이자 단점)
- 반복문 등과 함께 사용하면 아주 편리하게 데이터를 다룰 수 있음 ~> 단점이라면 조금 어려울 수 있음 (2차원 배열)
- 배열 이름은 여러 개의 데이터를 담는 것이라 복수형으로 사용하는 편 ~> 필수는 X

4. 배열 사용
- 일단 한 가지 방법만 있는 것이 아니고 여러가지 방법이 있음을 일단 기억
- 배열 사용 시 일단 알아야 할 것들
    1. 배열 선언 방법
    2. 선언한 배열에 초깃값 셋팅 방법 ~> 보통은 선언과 동시에 초깃값 셋팅
    3. 배열 요소에 접근하는 방법 ~> 요소, 항목, 아이템(item) 등등 다 비슷한 말 / 접근을 참조하고도 표현
    4. 배열에 데이터를 입력, 출력, 수정, 삭제 처리하는 방법
    5. 배열의 길이를 구하는 방법
    6. 반복문을 사용해서 배열의 요소를 순회하면서 출력시키는 방법 ~> 합을 구한다든지 등의 연산을 수행하는 방법

5. 배열 선언
int num;        누가봐도 정수형 변수 선언
int arr[];      arr 옆에 괄호 표시가 있으면 이것이 곧 배열 선언
                하지만, 이 단계는 배열을 arr 이름으로 선언만 한 것이고 아직 메모리 할당 등은 안되어진 상태 ~> new 해줘야함
                보통은 배열의 선언과 동시에 초깃값을 셋팅
String str[];   문자열 값을 담을 수 있는 배열을 선언 ~> 선언만 한 것이고 아직 크기는 정한게 없음

6. 배열 선언과 동시에 초깃값 셋팅
int arr[] = new int[3];         정수 값을 담는 배열을 arr 이름으로 선언, 정수는 3개(크기)를 담을 수 있음, 메모리 할당됨
String str[] = new String[5];   문자열 값을 담는 배열을 str 이름으로 선언, 문자열은 5개(크기)를 담을 수 있음, 메모리 할당됨

int arr[] = new int[]{1, 2, 3}; 변수 선언과 동시에 메모리 할당도 하고 값도 셋팅하는 방법1
int arr[] = {1, 2, 3};          변수 선언과 동시에 메모리 할당도 하고 값도 셋팅하는 방법2
                                보통은 간편한 이 방식을 많이 사용

String str[] = {"abc", "cd", "ef"};
