[ 큐 ]
- 선입선출 방식의 FIFO 방식으로 데이터 처리가 이루어지는 자료구조 형태
- FIFO : First In, First Out
- 쉽게 말해, 가장 먼저 저장된(push) 데이터가 가장 먼저 나오게(pop) 되는 구조
- 큐 자료구조 구현은 ArrayDeque 클래스를 사용하는 것도 가능
- 보통 큐 자료구조를 구현하는 경우  Deque 인터페이스를 구현한 LinkedList 클래스를 사용

[ 큐 예시 ]
- 관공서, 은행 등의 대기번호 표
- 맛집 대기줄, 프린터 대기열 등등...

[ 주요 메서드 ]
- add()         스택의 push() 메서드에 해당
- poll()        스택의 pop() 메서드에 해당, 큐가 비었으면 null 반환
- peek()        
- remove()
- element()
- offer()
