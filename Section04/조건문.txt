[1]
조건문 사용은 대부분의 프로그래밍 언어에서 비슷함
단지 프로그래밍 언어의 특징에 따라서 사용법이나 작성 규칙이 조금 다를 뿐
결론 : 한 번 배워두면 비슷한 부분이 굉장히 많아서 다른 프로그래밍 언어 배울 때 비교적 쉽다.

<제어문>
- 조건문 : (if / switch)
- 반복문 : (for / while / do ... while) 

[2]
조건문의 역할
- 조건에 따른 분기처리는 프로그래밍 작성에서 가장 기본이자 가장 많이 사용되는 문법적 요소

[3]
조건문 종류
- if
- if..else
- if..else..if
- switch
- 삼항연산자
- 단항 이항 삼항

[4]
switch 조건문
- if 조건문보다 조건을 깔끔하게(보기좋게) 적용
- 자동완성 기능을 사용하면 편리하게 기본 코드를 입력 
- 변수의 값 또는 어떤 계산식의 결과 값에 따라 조건을 적용하여 분기 처리하는 방식
- 변수의 값?
    - key 라고함 key-value 구문
    1. 숫자값 사용 O
    2. 문자열 사용 O
    3. 논리값 사용 X -> boolean -> true, false
    4. enum 타입 사용 O
- case 구문에서 함수(메서드) 호출도 가능
